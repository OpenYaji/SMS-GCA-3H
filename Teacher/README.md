### Installation

Follow these steps to set up the project locally:

#### **Step 1: Clone the Repository**

```powershell
# Navigate to your XAMPP htdocs directory
cd C:\xampp\htdocs

# Clone the repository
git clone https://github.com/tambayNgOrtigasAvenue/sia-teacher-side.git gymnazo-christian-academy-teacher-side

# Navigate to project directory
cd gymnazo-christian-academy-teacher-side
```

#### **Step 2: Backend Setup**

```powershell
# Navigate to backend directory
cd backend

# Install PHP dependencies
composer install

# Create environment file
cp .env.example .env

# Edit .env file with your database credentials
notepad .env
```

**Configure your `.env` file:**

```env
# Database Configuration
DB_HOST=localhost
DB_PORT=3306
DB_NAME=gymnadb
DB_USER=root
DB_PASS=

# Application Settings
APP_ENV=development
APP_DEBUG=true
APP_URL=http://localhost

# Email Configuration (PHPMailer)
MAIL_HOST=smtp.gmail.com
MAIL_PORT=587
MAIL_USERNAME=your-email@gmail.com
MAIL_PASSWORD=your-app-password
MAIL_ENCRYPTION=tls
MAIL_FROM_ADDRESS=noreply@gymnazo.edu
MAIL_FROM_NAME="Gymnazo Christian Academy"

# Session Configuration
SESSION_LIFETIME=3600
SESSION_SECURE=false

# Security
ENCRYPTION_KEY=your-32-character-encryption-key
```

#### **Step 3: Database Setup**

```powershell
# Start XAMPP MySQL service
# Open phpMyAdmin: http://localhost/phpmyadmin

# Import database schema
# 1. Create database named 'gymnadb'
# 2. Import: docker/mysql/init.sql
```

**Or use command line:**

```powershell
# Create database
mysql -u root -p -e "CREATE DATABASE gymnazo_db;"

# Import schema
mysql -u root -p gymnadb < docker\mysql\init.sql

# Set up teacher permissions
mysql -u root -p gymnadb < backend\setup-teacher-permissions.sql
```

#### **Step 4: Frontend Setup**

```powershell
# Navigate to frontend directory
cd ..\frontend

# Install npm dependencies
npm install

# Create frontend environment file
cp .env.example .env

# Edit frontend .env (if needed)
notepad .env
```

**Frontend `.env` configuration:**

```env
VITE_API_BASE_URL=http://localhost/gymnazo-christian-academy-teacher-side/backend/api
```

#### **Step 5: Start Development Servers**

**Terminal 1 - Frontend:**
```powershell
cd frontend
npm run dev
```

**Terminal 2 - Backend:**
```powershell
# Ensure XAMPP Apache and MySQL are running
# Backend is served via XAMPP at:
# http://localhost/gymnazo-christian-academy-teacher-side/backend
```
